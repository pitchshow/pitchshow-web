Core concepts

Spec Mode

The full AI pipeline: describe → research → storyline review → slide generation. PitchShow does the work; you approve the narrative before it builds.

Vibe Mode

Click any element on an existing slide and type what you want changed. Works with natural language — no need to write code or find the right menu.

Presenter Mode

Dual-screen presenter view with speaker notes, next slide preview, and rehearsal timer. Your audience sees the show; you see your notes.

Export Formats

PDF, PPTX (standard), Animated PPTX (Pro, server-side), HTML single-file, Speaker Notes markdown, and hosted show URL.
