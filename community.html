Contribute

The codebase is yours.

Apache 2.0. Fork it, modify it, ship it in your own product. If you build something worth merging back, we'll review PRs within 48 hours. Here's where we need the most help right now:

Good first issue

Export format improvements

PPTX export edge cases — fonts, nested layouts, complex animation sequences. If you care about presentation fidelity, this is the area where effort has the highest visible impact.

Browse issues
Wanted

Windows & Linux testing

We develop primarily on macOS. The Electron app runs on all platforms but we catch Windows and Linux regressions slower than we should. If you run either, installing and reporting what's broken is genuinely valuable.

Report an issue
Discussion

AI provider integrations

We support Bedrock, OpenAI, DeepSeek, Gemini, Groq, Grok. If your preferred provider isn't listed or behaves unexpectedly, open a Discussion. New provider support is usually straightforward to add.

Start a discussion
Before opening a PR for a large change, open a Discussion first. We'll tell you whether it aligns with where we're heading and save everyone time if it doesn't.
